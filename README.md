# Dashboard Analisis Pasar Mobil Bekas Jabodetabek

## Deskripsi

Proyek ini merupakan implementasi Business Intelligence untuk menganalisis pasar mobil bekas di wilayah Jabodetabek berdasarkan data iklan OLX.

Dashboard dikembangkan menggunakan Microsoft Power BI dan memanfaatkan metode K-Means Clustering untuk melakukan segmentasi kendaraan.

## Fitur Dashboard

- Jumlah Mobil Terdaftar
- Analisis Harga Mobil
- Analisis Merek Mobil Terbanyak
- Analisis Harga Berdasarkan Lokasi
- Analisis Jenis Bahan Bakar
- Tren Harga Berdasarkan Tahun
- Segmentasi Kendaraan Menggunakan K-Means Clustering

## Tools

- Microsoft Power BI
- Microsoft SQL Server
- Python
- K-Means Clustering
